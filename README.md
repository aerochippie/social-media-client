## To build this project
 1. Clone the repo
 2. Cd to location and type `code .`
 3. install dependencies : `npm i`
 4. build the app:
  `npm run build`
  `npm run buildApp`
  
 5. open with live server
  
 Open tests in cypress
 `npx cypress open`
